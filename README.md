# SMM Industrial Solutions SMC PVT Website

A modern, production-ready website built for SMM INDUSTRIAL SOLUTIONS SMC PVT. This application uses React Router, Framer Motion for animations, and TailwindCSS for beautiful styling.

## Features

- 🚀 Fast Server-side rendering
- ⚡️ Smooth Scroll Animations (Framer Motion)
- 📦 Fully Responsive Layout and modern Design
- 🎉 TailwindCSS for styling
- � TypeScript by default

## Getting Started

### Installation

Install the dependencies:

```bash
npm install
```

### Development

Start the development server with Hot Module Replacement (HMR):

```bash
npm run dev
```

Your application will be available at `http://localhost:5173`.

## Building for Production

Create a production build:

```bash
npm run build
```

## Previewing the Production Build

Preview the production build locally:

```bash
npm run preview
```

## Deployment

To deploy the application to your hosting provider, you first need to build it:

```sh
npm run build
```

Then you can use the deploy command:

```sh
npm run deploy
```
